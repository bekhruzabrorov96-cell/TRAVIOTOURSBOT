# Travio Tours Lead Bot Flow

## Main Goal

Client opens one link to the bot. The bot first collects name and phone number, then asks destination, number of travelers, client needs, and other tour questions. After that it sends a clean lead summary to the Travio Tours team.

## Bot Result For Client

Thank you. I found suitable tours and sent your request to a Travio Tours manager. They will contact you soon.

## Lead Sent To Manager

Name: client name  
Contact: phone number  
Destination: selected destination  
Travel date: selected date  
Travelers: number of people  
Needs: client needs and preferences  
Budget: selected budget  
Tour style: selected style  
Best matches: top 1-3 tour packages  

## Where To Use The Bot Link

- Instagram bio
- Website button
- Facebook page
- Telegram channel
- WhatsApp auto-reply
- Google Ads or Meta Ads lead campaign

## Recommended Real Setup

Telegram is the easiest first version because clients can open one link and the bot can send lead details directly to your manager group.

For a stronger business setup, connect the bot to:

- Google Sheets for all leads
- Telegram group for instant manager alerts
- CRM if Travio Tours already uses one
- Tour catalog spreadsheet for package matching
